Special considerations and designs

1. Control the default ball speed.

For the elders, the ball speed should not be too fast at the beginning for them to get accustomed to the pace of the game. So the default speed is 2 and it would be easy for them to get familiar with the rules.

2. The paddle speed is comfortable.

When controlled by keyboard, the paddle speed is depended on the speed of the ball. If the ball speed is high, the paddle speed will be high for a convenience operation.

3. Reduce the number words in instructions.

I reduce the words in instructions and use two replaceable pictures to introduce the layout and elements. It is much convenience for the elder to recognize each element.

4. Reduce the influence of changing elements.

The time is a changing element during the game period. Any changing element will affect user's focus. Therefore, I chose to use a relatively small font size to display the time.

5. Auto set the focus to canvas.

The game will set the key focus to canvas automatically during the game initialization in case of losing control as old people may not know the solution.
